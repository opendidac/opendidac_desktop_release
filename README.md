# opendidac_desktop_release
Cross-platform OpenDidac Desktop app for evaluations and consultations.

## Download OpenDidac Desktop

Choose your operating system to download the desktop application.

### ⚠️ Important: Authentication Requirements

Since you will be using the desktop application, your browser's saved passwords will not be available. Make sure you have:

- Your Switch edu ID username and password
- Access to your authenticator app

---

## Windows

**Download:** [OpenDidac.Desktop-1.0.0.Setup.exe](https://github.com/opendidac/opendidac_desktop_release/releases/download/v1.0.0/OpenDidac.Desktop-1.0.0.Setup.exe)

### Installation Instructions:

1. Download the Windows installer (.exe file)
2. Run the installer
3. If Windows Defender displays "Windows protected your PC", click on "More info" and then "Run anyway"
4. The setup will install and run the application automatically.
5. You can use the special link, in your browser, provided by your professor to access the evaluation

## macOS

### Download:

- **Apple Silicon (M1, M2, M3, etc.) - ARM64:** [OpenDidac.Desktop-darwin-arm64-1.0.0.zip](https://github.com/opendidac/opendidac_desktop_release/releases/download/v1.0.0/OpenDidac.Desktop-darwin-arm64-1.0.0.zip)
- **Intel Mac - x64:** [OpenDidac.Desktop-darwin-x64-1.0.0.zip](https://github.com/opendidac/opendidac_desktop_release/releases/download/v1.0.0/OpenDidac.Desktop-darwin-x64-1.0.0.zip)

### Installation Instructions:

1. Download the macOS application (.zip file) for your Mac type:
   - Apple Silicon (M1, M2, M3, etc.) - ARM64 version
   - Intel Mac - x64 version
2. Extract the ZIP file to your Applications folder
3. Open Terminal and run the following command (replace with your actual path):
   ```
   xattr -d com.apple.quarantine "/Applications/OpenDidac Desktop.app"
   ```
4. If you see a security warning, go to System Preferences > Security & Privacy and click "Open Anyway"
5. Launch OpenDidac Desktop from your Applications folder
6. You can use the special link, in your browser, provided by your professor to access the evaluation
